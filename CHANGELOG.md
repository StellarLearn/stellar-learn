# Changelog

## v0.1.0 — Initial Scaffolding

- Monorepo setup with Turborepo (apps/web, apps/docs, apps/contracts, packages/)
- Soroban smart contracts scaffolded: ProofOfLearning and RewardPool
- Frontend foundation: landing page hero, Navbar, Footer, CourseCard components
- API routes: /api/courses and /api/health
- Shared UI component library with Button and Card
- Documentation: README, CONTRIBUTING, ROADMAP, ISSUES, SECURITY, architecture docs
- CI workflows for build verification and linting
- GitHub issue templates, PR template, and CODEOWNERS
- TypeScript type definitions for Course, User, Quiz, Reward, Certificate
- Stellar SDK and Soroban RPC client utilities
- React hooks: useWallet and useCourses
- CSS design tokens and dark mode support
